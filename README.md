# Магазин товаров

Данный проект был выполнен в качестве тестового задания и содержит в себе одностраничное веб-приложение с поиском товаров и модальным окном просмотра информации о товаре. 
Проект был написан на **HTML, CSS и JavaScript**.

## Инструкция по запуску

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/wanychto/shop-app
   cd <НАЗВАНИЕ_ПРОЕКТА>```
   
2. **Рекомендуется запуск локльного сервера для корректной работы fetch(data.json). Если установлен Python, в терминале пишем:**
   ```python -m http.server```
   
3. **Откройте в браузере**
   ```http://localhost:8000```
